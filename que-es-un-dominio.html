¿Qué es un dominio?

Un dominio es el nombre único y exclusivo que se le da a un sitio web en Internet para que cualquiera pueda visitarlo.

Los navegadores web acceden a los sitios en Internet por medio de una dirección IP (Internet Protocol), similar a esta: 104.28.13.62 que sirve para saber en qué servidor se encuentran.
Utilizar un nombre dominio en un sitio web en vez de la dirección IP tiene por lo menos 3 grandes ventajas:
    1.-  El dominio de un sitio es mucho más fácil de recordar que su IP.

   2.-Permite tener más sitios en Internet en el mundo debido a que muchos dominios web pueden compartir la misma IP del servidor web donde están alojados (luego internamente el servidor sabe a qué dominio se está accediendo).

    3.- Es mucho más flexible por ejemplo a la hora de cambiar tu sitio de un alojamiento web a otro, porque la IP puede cambiar pero el dominio seguirá siendo el mismo.
